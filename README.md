# 3D 抽奖应用 (原生 JS 版)

这是一个高性能、容器化的 3D 抽奖应用程序，使用纯 HTML、CSS、JavaScript 和 Three.js 构建。无需构建步骤。

## 如何运行

1. **启动应用**:
   ```bash
   docker compose up
   ```

2. **访问应用**:
   打开 [http://localhost:3000](http://localhost:3000)

## 功能特点
- **3D 旋转云**: 在 3D 球体中展示参与者。
- **Excel 导入**: 支持包含 `Name` (姓名), `Phone` (电话), `Department` (部门) 列的 `.xlsx` 文件。
- **抽奖逻辑**: 带有排除逻辑的随机选择。
- **毛玻璃 UI**: 现代透明界面风格。
- **数据持久化**: 自动保存到 LocalStorage。
- **可自定义**: 添加奖项，更改背景图片。

## 验证步骤
1. **打开** URL。
2. **点击 "测试数据"** 生成参与者。
3. **选择奖项**: 从左侧边栏选择一个奖项。
4. **点击 "开始"** 开始旋转。
5. **点击 "停止"** 选取中奖者。
6. **查看名单**: 点击 "中奖名单" 按钮查看列表。

## 项目结构
- `frontend/`: 包含所有静态代码。
  - `index.html`: 入口文件。
  - `css/style.css`: 样式文件。
  - `js/`: 逻辑模块 (`main.js`, `lottery3d.js`, `data.js`)。
- `docker-compose.yml`: 服务定义。
